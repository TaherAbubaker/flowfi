# 💰 FlowFi

**Your money, in flow.**

FlowFi is a modern Android personal finance application designed to help users track income, expenses, savings goals, and financial habits in a clean and minimal dark-themed interface.

---

## 📱 Features

- 📊 Track income and expenses
- 💸 Categorized transactions
- 🎯 Savings goals with progress tracking
- 🔥 Daily financial challenges (streak system)
- 📈 Analytics dashboard (charts & insights)
- 💬 Simple finance chatbot (keyword-based assistant)
- 🌙 Modern dark UI with green accent theme
- 🧾 Transaction history with filtering & search

---

## 🎨 UI Design

FlowFi uses a clean dark theme:

- Background: `#121212`
- Cards: `#1E1E1E`
- Accent: `#00C853` (emerald green)
- Expense color: `#FF5252`
- Income color: `#00C853`

Typography uses Android system fonts (no external font dependency).

---

## 🧱 Tech Stack

- Java (Android)
- XML Layouts
- SQLite Database
- RecyclerView
- MPAndroidChart (for analytics)
- Material Design Components
